## Getting Started

A powerful JavaScript &amp; TypeScript framework for faster and easier bot creation.

- Visit our [Documentation](https://adbjs.vercel.app/docs) to get started with ADB.js.

### Installation

```bash
npm install adb.js
pnpm add adb.js
yarn add adb.js
bun add adb.js
```

## Community

The ADB.js community can be found on [GitHub Discussions](https://github.com/adbjs/adb.js/discussions) where you can ask questions, voice ideas, and share your projects with other people.

To chat with other community members you can join the ADB.js [Discord](https://discord.gg/ZmKUnugG6s) server.

Do note that our [Code of Conduct](https://github.com/adbjs/.github/blob/main/CODE_OF_CONDUCT.md) applies to all ADB.js community channels. Users are **highly encouraged** to read and adhere to it to avoid repercussions.

For more information, visit our [Community Support](https://github.com/adbjs/.github/blob/main/SUPPORT.md).

## Contributing

Contributions to ADB.js are welcome and highly appreciated. However, before you jump right into it, we would like you to review our [Contribution Guidelines](https://github.com/adbjs/.github/blob/main/CONTRIBUTING.md) to make sure you have a smooth experience contributing to ADB.js.

## Security

If you have any reason to believe that there are security vulnerabilities across
any repository in the organisation, please report to
**[adbjs.contact@gmail.com](mailto:adbjs.contact@gmail.com)**.

We will aim to respond as quickly as possible, but note that the above mailbox
is **not** monitored 24/7. If the report is confirmed to be true, we will make
the necessary changes required to fix the issue.
